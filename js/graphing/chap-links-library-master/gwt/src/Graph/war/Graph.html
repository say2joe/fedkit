Links Graph

This Graph Entry Point is for test purposes. To run the test program, you have to open the file Graph.gwt.xml, and uncomment the line

<entry-point class='com.chap.links.client.GraphEntryPoint'/>

such that the entry point is defined.

If you want to create a stand-alone module for the graph, you have to comment the line with the entry point again, and then create a jar containing only Graph.java, Graph.gwt.xml, graph.js, graph.css, and excanvas.js.
