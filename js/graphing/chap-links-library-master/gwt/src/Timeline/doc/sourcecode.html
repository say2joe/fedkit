Timeline GWT sourcecode documentation

The timeline is written in pure Javascript. There is a GWT wrapper written that enables you to use the Timeline from within GWT (Google Web Toolkit). This page explains how the sourcecode is built up. For detailed information about all functions, one can use the JavaDoc reference pages.

Creation of JAR module

In order to put the Timeline files in one jar module:

  • Comment the line with the entry point in Timeline.gwt.xml
  • Right click on the project, choose Export, "Jar file".
  • Select the following files and folder
    • src/com.chap.links/Timeline.gwt.xml
    • src/com.chap.links/public/timeline.js
    • src/com.chap.links.client/Timeline.java
    • src/META-INF
  • Select two options:
    • Export generated class files and resources
    • Export Java source files and resources
  • Select an output filename and location
  • Click Finish
